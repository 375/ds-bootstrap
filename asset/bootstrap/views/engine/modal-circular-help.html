Reading circulars


ads Flyer view

On this view, mouse over or tap an item on the image/page to display interactive buttons. Click/tap the button to see more information or add item to list. Click the pagination buttons to change page. With a touch enabled device, you can also swipe left/right on the image to switch between pages.

Non-interactive flyer will be denoted as: * image only/non-interactive flyer


ads List view

This view allow you to quickly sort and search for sale items that are available on the flyer. Click or tap on the item description to see more information.


ads Text-only view

The has all function of list view but without image. This is friendly with user that has no use for image or is on slower connection or is has low on mobile bandwidth.
